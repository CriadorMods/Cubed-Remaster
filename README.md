## üöÄ Getting Started  

### 1. Clone the Repository  
```
git clone https://github.com/CriadorMods/Cubed-Remaster.git
cd Cubed-Remaster
```

### 2. Install Dependencies  
```
bun i
```

### 3. Run the Development Server  
```
bun vite
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.
